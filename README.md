# Flight Booking Service Demo

----------

###### Getting Started

pipenv

`pip install pipenv`

goto directory

`cd FlightAssist`

install dependencies

`pipenv install`

start shell

`pipenv shell`

start the project

`python main.py`

or if you can directly run without shell

`pipenv python main.py`

Note: Make sure to have python 3.7 or above
